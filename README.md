# Fitness Tracker Component

A comprehensive fitness progression tracker allowing users to create workout schemas, log weights, reps, and sets, and visualize progress.

## Features
- **Dashboard**: Overview of recent activity and quick start.
- **Schema Builder**: Create custom workout routines.
- **Workout Logger**: Track sets, reps, and weights in real-time.
- **History**: View past workout logs.
- **Responsive**: Mobile-first design that scales to desktop.

## Usage

```tsx
import { FitnessTracker } from '@/sd-components/72070ca8-8a73-49a2-8d94-3d70fc369285';

function App() {
  return <FitnessTracker />;
}
```
